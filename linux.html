代码笔记

压缩和解压

操作 命令 说明
压缩文件夹 
cd MPOS/
zip -r data.zip data/
 将 data 文件夹及其内容递归地压缩成 data.zip 文件。
压缩文件夹指定路径 
zip -r data.zip /home/myj/MPOS/data/
 将 data 文件夹及其内容递归地压缩成 data.zip 文件。
解压缩文件 
unzip data.zip
 将 data.zip 文件解压缩到当前目录。

登录到FTP服务器

操作 命令 说明
登录到FTP服务器 
ftp example.com
 连接到FTP服务器 example.com。之后会提示输入用户名和密码。如果连接的是ip地址 则是ftp 192.168.123.16 2121
登录到FTP服务器 
ftp
o 192.168.123.16 2121
 连接到FTP服务器 example.com。之后会提示输入用户名和密码 , 如果是ip则空格加上端口。
上传文件 
put data.zip
 将本地文件 data.zip 上传到当前远程目录。
下载文件 
get data.zip
 将远程文件 data.zip 下载到本地当前目录。也就是home
下载文件到这个目录下 
get data.zip /home/myj/MPOS/data.zip
 将远程文件 data.zip 下载到MPOS目录下。
列出目录内容 
ls
 列出当前远程目录的内容。
退出FTP会话 
bye
 退出FTP会话。

查看命令

操作 命令 说明
查看运行路径 
ps -ef | grep vino-server
 查看指定进程的运行路径。
查找指定进程 
ps aux | grep gnome-terminal
 查找包含gnome-terminal的进程。
查看内存占用情况 
htop
 交互式地查看系统的资源使用情况。
查看文件路径 
locate <文件名>
 查看这个文件所在的路径
查看运行情况 
top
 显示系统的实时运行情况，按m键按内存使用排序。
查看系统当前内存使用情况 
free -h
 以人类可读格式显示内存使用情况。
查看磁盘使用情况 
df -h
 显示磁盘空间使用情况。
查看系统位数 
getconf LONG_BIT
 显示系统架构的位数。
查看CPU架构 
uname -m
 显示系统的CPU架构。
查看IP 
ip a s
ifconfig
hostname -I
 显示网络接口的详细信息。
查看hosts文件内容 
cat /etc/hosts
 显示hosts文件的内容。

文件操作

操作 命令 说明
解压tar.gz文件 
tar -zxvf frp_0.52.0_linux_amd64.tar.gz
 解压指定的tar.gz文件。
列出目录内容 
ls -l
 以长格式列出当前目录的内容，包括文件权限、所有者、文件大小和修改日期。
列出目录内容 
dir
 列出当前目录的内容
创建文件 
touch 1.txt
 创建一个新的空文件。
创建多个文件 
touch 1.txt 2.txt
 创建多个新的空文件。
删除文件 
rm -rf （文件）
 删除指定的文件或目录。
创建文件夹 
mkdir myj
 创建一个目录。
删除文件夹 
rmdir myj
 删除一个目录。
移动文件 
mv nano.save /home/myj/MPOS/
 当前目录下的nano.save移动到MPOS目录。
重命名文件 
mv nano.save nane.txt
 nano.save 重命名为nane.txt
复制文件到/tmp目录 
cp frp_0.52.0_linux_amd64/frpc.ini /tmp
 将frpc.ini文件复制到/tmp目录。
复制文件夹 
cp -r flie /home/myj/
 复制文件夹及内容到这个目录
解压tar.gz文件 
tar -zvxf 1.tar.gz
 解压指定的tar.gz文件。
查看文件内容 
cat frps.ini
 显示文件的内容。

权限命令

操作 命令 说明
给全部子目录及文件权限改为777 
sudo chmod 777 * -R
 递归地更改当前目录下所有文件和子目录的权限为777。
获取root权限 
sudo -i
 切换到root用户。
给hosts文件所有权限 
sudo chmod 777 /etc/hosts
 更改hosts文件权限。
设置root密码 
sudo passwd root
 更改或设置root用户的密码。
给文件所有权限 
chmod +x frpc.ini
 更改文件权限为可执行。
杀死全部进程 
sudo pkill MPOS
 杀死所有名为MPOS的进程。

可能用到的命令

操作 命令 说明
立即关机 
shutdown -h 0
 立即关闭系统。
立即重启 
shutdown -r 0
 立即重启系统。
5秒后执行重启命令 
sleep 5 && sudo shutdown -r now
 延迟5秒后重启系统。
initramfs下重启 
reboot
 立即重启系统。
修复系统开机进入initramfs 
fsck -t ext4 /dev/sda1
 检查并修复文件系统错误（输入y确认）。
启用有线连接 
sudo nmcli networking on
 启用网络连接。
禁用有线连接 
sudo nmcli networking off
 禁用网络连接。
打开火狐浏览器 
firefox
 启动Firefox浏览器。
切换到图形界面 
ctrl+alt+F7
 切换到图形用户界面。
进入tty1命令行界面 
ctrl+alt+F1
 切换到第一个虚拟控制台。
下载mpos文件并重命名为1.zip 
wget -O 1.zip 1.admintop.top
 从指定URL下载文件并重命名。
修复系统开机进入initramfs 
fsck -t ext4 /dev/sda1
 检查并修复文件系统错误（输入y确认）。
进入桌面 
ctrl + alt + d
 切换到桌面。
打开终端 
ctrl + alt + t
 启动终端。
进入tty1，F7进入图形化界面 
ctrl + alt + F1
 切换到第一个虚拟控制台。
启动frpc 
/tmp/frpc -c /tmp/frpc.ini
 使用指定的配置文件启动frpc。
安装apt 
sudo apt-get install apt
 安装APT包管理器。
更新和升级系统并安装x11vnc 
sudo apt update && sudo apt upgrade -y && sudo apt install x11vnc -y
 更新和升级系统包并安装x11vnc。
查看SSH登陆账号 
whoami
 显示当前登录的用户名。
在vi中搜索 
:/fun
 在vi编辑器中搜索"fun"。
重启SSH连接 
sshd &
 在后台重启SSH服务。

总结

在这份笔记中，我们记录了一些常用的命令和它们的使用方法。这些命令包括压缩和解压缩、文件操作以及查看系统信息。
