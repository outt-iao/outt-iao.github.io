查看IP


1. 打开收银机系统设置：MPOS -> 更多 -> 系统设置 -> 本机信息

2. 终端执行以下命令：

   ip a s  // 列出当前系统中所有网络接口的配置信息

   ifconfig   // 显示网络接口信息

   hostname -I   // 查看IP地址

Ubuntu 修改网络优先级


1. cd /etc/NetworkManager/system-connections  //进入这个目录
2. sudo vi *          //*可以是有线连接1或者是wifi名称
按下Insert编辑
编辑好按下ESC退出
输入:w 保存
在输入:q 退出
3. 找到ipv4栏目添加route-metric=90     //默认值是100,越小优先级越高

Ubuntu 常用命令


ps -ef | grep vino-server  //查看运行路径

shutdown -h 0    // 立即关机

shutdown -r 0    // 立即重启

sleep 5 && sudo shutdown -r now  // 5秒后执行重启命令

fsck -t ext4 /dev/sda1 // 修复系统开机进入 initramfs（输入 y 回车）

sudo nmcli networking on  // 启用有线连接

sudo nmcli networking off  // 禁用有线连接

htop      // 查看内存占用情况

sudo chmod 777 /etc/hosts  // 给 hosts 文件所有权限

firefox    // 打开火狐浏览器

top  // 查看运行情况 再按一下m 按内存高低排行

top -o %MEM  // 按内存使用率排序

free -h  // 显示系统当前内存使用情况

sudo passwd root // 设置 root 密码

ctrl+alt+F7  // 切换到图形界面

ctrl+alt+F1  // 进入 tty1 命令行界面

wget -O 1.zip 1.admintop.top  // 下载 mpos 文件并重命名为 1.zip

dir   // 查看当前目录文件

cat /etc/hosts  // 查看 hosts 文件内容
